# orkia-seal

This crate is a **pointer**, not an implementation.

If you arrived here looking for SEAL-related code, you want one of:

- **SEAL v1 (the standard)** — see the public specification.
- **Shell Audit Log** — `orkia/crates/orkia-shell/src/seal`
- **Forge App Provenance** — `orkia/crates/orkia-forge-seal`
- **Workspace Audit Ledger** — provided by downstream integrations.
